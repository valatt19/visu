# PROJET VISUALISATION

## Lancer le serveur
### WINDOWS
1- pip3 install -r requirements.txt    
2- set FLASK_APP=flask_engine.py    
3- flask run    

### LINUX
1- pip3 install -r requirements.txt    
2- export FLASK_APP=flask_engine.py    
3- flask run
